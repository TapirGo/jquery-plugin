h1. TapirGo jQuery plugin

A plugin that makes searching with TapirGo a breeze!

Live example at "http://tapirgo.com/examples/jquery-plugin/index.html":http://tapirgo.com/examples/jquery-plugin/index.html

h2. Using the plugin

1. Make sure you have jQuery in your site (any version above 1.2 should work).
2. Add the jquery-tapirgo.min.js to your site.
3. Call tapirgo() on the element you want the results to appear in.

example:

<pre><code>
  <!DOCTYPE html>

  <html>
    <head>
      <meta charset="utf-8">
      <title>TapirGo jQuery plugin example</title>
    </head>

    <body>

      <div id="search_results"></div>

      <script src="jquery-1.6.1.min.js"></script>
      <script src="jquery-tapirgo.min.js"></script>
      <script>
        $('#search_results').tapirgo({'token': '4dbfc79e3f61b05b53000011'});
      </script>

    </body>
  </html>
</code></pre>
